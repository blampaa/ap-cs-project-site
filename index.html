Drawing App

Link to App

The drawing app project is quite simple. The user is asked how many sides the shape should have, and the character on screen will draw their answer as a shape. So, if the user responds with “8 sides,” then, the character on screen will draw an octagon. And so on. It works with any number of sides. If the user responds with “1” or “2” sides, then, there will be an error, and the code will ask for a different number.

Instagram Macro

This software is actually intended to provide value to small businesses and make money. First, it needs a list of usernames for it to engage with for it to work. How it works: it will go to each username, follow them, and like their pictures. After 2 days, it will automatically unfollow the people they followed. It is all automatic and can run every day. This is very helpful for small businesses because all they need is a list of usernames that are their niche customers and could be potential leads. Then, the software works and potentially increases their sales.
